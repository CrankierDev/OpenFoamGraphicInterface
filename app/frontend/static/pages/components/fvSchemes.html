Esquemas predeterminados de la simulación

info

Esquemas aplicados a las derivadas temporales para el cálculo de las ecuaciones. Se han definido dos opciones:

  • Esquema estacionario: Las derivadas de tiempo en las ecuaciones son despreciadas.
  • Esquema de Euler: Método de primer orden con el que las derivadas de tiempo se calculan a partir de un valor inicial.

Esquema para los gradientes

info

Esquemas aplicados en el cálculo de gradientes. Estos esquemas se subdividen en dos partes: esquema e interpolación; y se dan las siguientes opciones:

  • Para el esquema:
    • Teorema de Gauss
    • Mínimos cuadrados
  • Para la interpolación:
    • Lineal basado en las celdas
    • Lineal basado en puntos
    • Mínimos cuadrados

Esquema para las divergencias

info

Esquemas aplicados en el cálculo de divergencias. Estos esquemas se subdividen en tres partes: delimitación, esquema e interpolación. Se utiliza el teorema de Gauss como esquema, el esquema se puede o no delimitar.y para la interpolación, se dan las siguientes opciones:

  • Lineal
  • Lineal limitado
  • Lineal aguas arriba
info

Esquemas aplicados en el cálculo de laplacianos. El esquema usado es el teorema de Gauss. También se define por defecto el uso de la interpolación lineal. Otra característica, que se deja a elección del usuario es esquema para los gradientes normales a la superficie. Las opciones para este último parámetro son:

  • Corregido
  • Ortogonal
